# Public Package Notes: paper-2015-03

The paper-stated raw data are public Census/TIGER road-network GIS files and Census-defined MSA geographies. The local folder contains derived workbook evidence for circuity, speed, accessibility, and regression summaries. These local derived workbooks are the archival target; raw agency GIS files should be cited through public source links rather than duplicated.

The package includes workbook originals plus CSV exports for inspection. The originals preserve formulas and workbook structure; the CSV exports are convenience copies and should not be treated as a complete computational workflow. No ArcGIS/Network Analyst scripts were found.
